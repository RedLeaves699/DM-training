## 5.1-1d:
N(x):x为自然数\
S(x,y):y是x的后继\
E(x,y):x==y\
($\forall$x) ( N(x)$\rightarrow (\exists$y) ( S(x,y)$\wedge$ ($\forall$z) ( S(x,z) $\rightarrow$ E(y,z) ) )
## 5.1-2b:

$\neg$($\exists$x)( R(x)$\wedge$ Odd(x) $\wedge$ Even(x) ) )
